# Basic Symfony Example

This workspace demonstrates TwigPlus with static Symfony metadata v3.

- `path('...')` and `url('...')` complete route names.
- `asset('...')` completes known asset paths.
- Translation strings used by `trans()` or the `trans`/`t` filter complete known message ids.
- No PHP, Composer autoloader, Symfony Console command, or application code is executed by the extension.
